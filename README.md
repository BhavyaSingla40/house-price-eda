\# House Price Exploratory Data Analysis (EDA)



\##  Objective

The objective of this project is to explore and analyze the factors that influence house prices using the California Housing dataset.  

The project focuses on data cleaning, exploratory data analysis, feature engineering, and visualization to extract meaningful insights.



---



\##  Dataset

\- \*\*Name:\*\* California Housing Dataset  

\- \*\*Source:\*\* scikit-learn  

\- \*\*Description:\*\*  

&nbsp; The dataset contains housing-related information such as median income, house age, location, population, and median house value for districts in California.



---



\##  Tools \& Technologies Used

\- Python

\- Pandas

\- NumPy

\- Matplotlib

\- Seaborn

\- Scikit-learn



---



\##  Project Workflow

1\. Data loading and inspection  

2\. Handling missing values  

3\. Feature engineering  

4\. Exploratory Data Analysis (EDA)  

5\. Correlation analysis  

6\. Data visualization  

7\. Insight generation  



---



\##  Key Analysis Performed

\- Distribution analysis of house prices  

\- Correlation analysis between numerical features  

\- Relationship between median income and house value  

\- Geographic visualization of house prices using latitude and longitude  

\- Comparison of house prices based on proximity to the ocean  



---



\##  Key Insights

\- Median income shows a strong positive correlation with median house value  

\- Houses located near the ocean generally have higher prices  

\- Geographic location plays a significant role in determining house prices  

\- House age has a weaker influence compared to income and location  



---



\##  Conclusion

This project demonstrates an end-to-end data analysis workflow, including data preprocessing, exploratory analysis, and visualization.  

It highlights how statistical analysis and visual exploration can be used to understand real-world datasets and derive actionable insights.
